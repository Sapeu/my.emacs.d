# 学习 emacs 的记录

## 下载和使用

``` shell
# 若已有.emacs.d/文件夹, 还请考虑备份后再下载
git clone git@github.com:Sapeu/my.emacs.d.git ~/.emacs.d
```

正常启动`emacs`即可
